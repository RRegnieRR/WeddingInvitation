# Sitio de Boda de Regnier & Alizee

Luxury romantic wedding microsite built with Vite, React, Tailwind CSS, and Framer Motion.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

4. Preview production build:

```bash
npm run preview
```

## Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
